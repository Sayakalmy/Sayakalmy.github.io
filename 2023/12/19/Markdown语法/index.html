Markdown 基础语法

简介：md格式是近几年流行的纯文本格式，用md格式来写说明文档或者blog十分的方便，目前github和CSDN都支持用md格式来写blog，本文主要记录markdown格式的基本语法，快捷键在一些软件环境下适用（本文编写使用的是Typora），其他语法在所有支持md格式的软件都适用。

Markdown常用快捷键

标题：

一级标题ctrl+1 ，二级标题crtl+2，三级标题crtl+3;当用#号表示时，# 一级标题，## 表示二级标题，依次类推

# 这是一个一级标题 ## 这是一个二级标题

具体呈现形式如下：

这是一个一级标题

这是一个二级标题

加粗：

快捷键 Ctrl + B将部分文字加粗显示：这是加粗，常规语法写法为**这是加粗**。

斜体：

快捷键 Ctrl + I将部分文字斜体显示：这是斜体，常规语法写法为*这是斜体*。

斜体并加粗：

这是斜体加粗，常规语法写法为***这是斜体加粗***。

下划线：

快捷键 Crtl + U将部分文字加下划线：这是下划线，常规语法写法为<u>这是下划线</u>。

代码块：

快捷键 Crtl + Shift + `将部分文字转换为代码：这是代码，常规语法写法为这是代码 。

删除线：

快捷键 Alt + Shift +5将部分文字加删除线：这是删除线，常规语法写法为--这是删除线--。

超链接：

快捷键 Ctrl + K将部分文字加超链接：这是超链接,常规语法写法为[这是超链接](链接地址)。

引用：

常规语法写为> 我是引用，表现形式如下：

我是引用

分割线：

常规语法写为***，表现形式如下：

任务列表：

常规语法写为- 我是任务列表，表现形式如下：

  • 我是任务列表
表情：

:sweat_smile:,写法为:表情（就喜欢流汗黄豆捏）

图片：

常规语法写为![图片名称](图片路径)，但是真有人会这么加图片吗，一般直接把图片拖到Typora中就行，或者右键直接插入。

Typora快捷键总结

快捷键 作用 快捷键 作用
Ctrl+1 一阶标题 Ctrl+I 字体倾斜
Ctrl+2 二阶标题 Ctrl+U 下划线
Ctrl+3 三阶标题 Ctrl+Home 返回Typora顶部
Ctrl+4 四阶标题 Ctrl+End 返回Typora底部
Ctrl+5 五阶标题 Ctrl+T 创建表格
Ctrl+6 六阶标题 Ctrl+K 创建超链接
Ctrl+L 选中某句话 Ctrl+F 搜索
Ctrl+D 选中某个单词 Ctrl+H 搜索并替换
Ctrl+E 选中相同格式的文字 Ctrl+Shift+I 插入图片
Alt+Shift+5 删除线 Ctrl+B 字体加粗

除了以上特殊语法外，Typora还支持常规的html语法（前端人的福音），在超链接及图片插入中运用html写法可以玩出很多花头（bushi），例如鼠标悬停在图片上显示神秘字符等等。总而言之，markdown作为一种轻量级语言，大大提升了文档的编辑效率，而Typora作为我个人认为的最好用的md文档编辑软件，十分值得大家学习。
